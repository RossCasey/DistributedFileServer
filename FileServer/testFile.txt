

Toolbar: CMD + SHIFT + P

Search: CMD + P
Search Symbol In File: CMD + R (or type @ after using CMD + P)
Search Symbol In Entire Project: CMD + SHIFT + R

Advanced Create New File: CMD + ALT + N

Change The Number Of Panes: ALT + CMD + 1/2/3/4/5

Select Current Word: Click into word and CMD + D
Holding CMD and Pressing D again will cause the creation of multiple cursors
for the next iteration of that word. 
To get all references and modify them: Control + CMD + G
